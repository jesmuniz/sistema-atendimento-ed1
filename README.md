# Sistema de Atendimento - ED1

## Objetivo

Sistema de gerenciamento de atendimentos utilizando estruturas de dados clássicas.

## Funcionalidades

- Cadastro de clientes
- Filas de prioridade
- Histórico
- Relatórios CSV
- Busca binária
- Pilha desfazer

## Estruturas utilizadas

- Filas
- Pilhas
- Lista encadeada
- Vetor ordenado
- Merge sort

## Como executar

python main.py
